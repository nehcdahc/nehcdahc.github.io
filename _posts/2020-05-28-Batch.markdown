## %cd% 与 %~dp0% 的区别

1. 使用范围

   - %cd%：.bat file、cmd
   - %~dp0%: .bat file

1. 示例脚本

   1. 创建文件 D:\cd~dp0.bat

      ```batch
      @echo off
      echo this is %%cd%% : %cd%
      echo this is %%~dp0 : %~dp0
      ```

   1. 在 C:\Users\Administrator 路径执行

      ```
      this is %cd% : C:\Users\Administrator
      this is %~dp0 : D:\
      ```

   1. 在 D:\ 路径执行

      ```
      this is %cd% : D:\
      this is %~dp0 : D:\
      ```
